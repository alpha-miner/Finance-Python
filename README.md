纯python实现的金融计算库，目标是提供进行量化交易必要的工具，包括但不限于：定价分析工具、技术分析指标。其中部分实现参考了quantlib。

依赖
-------------

    coverage
    enum34
    numpy
    pandas
    scipy

安装
-------------

首先将代码保存至本地：

    git clone https://github.com/ChinaQuants/Finance-Python.git (如果你是从github获取)
    cd finance-Python
    python setpy.py install


安装之后，可以直先接运行测试：

    python setup.py test
